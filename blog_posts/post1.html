What is HTML?

HTML stands for Hyper Text Markup Language. It is a formatting language used to define the appearance and contents of a web page. It allows us to organize text, graphics, audio, and video on a web page. It is not a programming language, but rather a language that identifies the meaning, purpose, and structure of text within a document. HTML has come a long way since Tim Berners-Lee invented it in 1991. HTML5 is the latest version supported by modern web browsers.

Hypertext The word Hypertext refers to the text which acts as a link.
Markup The word markup refers to the symbols that are used to define structure of the text. The markup symbols tell the browser how to display the text and are often called tags.
Language The word Language refers to the syntax that is similar to any other language.

Basic Requirements

In order to write HTML code, only a notepad and a browser is enough. But to have better visual and colorful experience, you may use a text editor like Sublime Text, Notepad++ or any other text editor of your choice. For web browser, Google chrome is preferred as it offers a lot of tools.

HTML Tags

Tag, also called elements, is a command that tells the web browser how to display the text, audio, graphics or video on a web page. Tags are indicated with pair of angle brackets. They start with a less than (<) character and end with a greater than (>) character. The tag name is specified between the angle brackets. Most of the tags usually occur in pair,i.e. the start tag and the closing tag. The start tag is simply the tag name enclosed in angle bracket whereas the closing tag is specified including a forward slash (/). Some tags are empty i.e. they don’t have the closing tag like the null tags. Tags are case insensitive. The starting and closing tag name must be the same. If you don’t specify the angle brackets for a tag, the browser will treat the tag name as a simple text.

HTML syntax

All HTML documents must start with a document type declaration. Then it begins and ends with html tags. The part which are visible on web pages is between the body tags of a HTML document.
